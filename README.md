# Employee & Department Analytics Dashboard

This interactive Power BI dashboard provides a comprehensive analysis of employee and departmental data.

## 📊 Dashboard Features:
- **Total Active Employees:** 100
- **Average Performance Score:** 3.43
- **Average Attendance Rate:** 87.08%
- **Visuals Included:**
  - Count of Employees by Department
  - Performance Scores by Department
  - Salary Distribution by Department (Treemap)
  - Gender-wise Employee Split
  - Experience vs Performance Scatter Plot
  - Attendance Trends Over Time

## 🎯 Purpose:
To help HR and management gain insights into employee productivity, diversity, attendance, and salary trends across departments.

## 🛠️ Tools Used:
- Power BI  
- Data Cleaning (Power Query)  
- Custom Visuals & DAX Calculations
